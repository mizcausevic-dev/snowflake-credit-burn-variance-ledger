# Architecture

The ledger has three layers:

1. `fixtures/credit_burn_variance.json` models Snowflake warehouse burn lanes.
2. `snowflake_credit_burn_variance_ledger/ledger.py` scores credit delta, idle time, failed query rate, unlabeled query rate, and auto-suspend gaps.
3. `sql/credit_burn_variance_contract.sql` describes the ACCOUNT_USAGE extraction contract.

The static site is generated by `scripts/prerender.py` and served from GitHub Pages.
